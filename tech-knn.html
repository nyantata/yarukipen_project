TECH CARD / LAB NOTE

k近傍法カード

近くにいる先輩データから予想する

このカードの出番

  • ●似ている過去データから状態を予想したい
  • ●分類のしくみを直感的に理解したい
  • ●難しいモデルの前に分類を試したい

作れるもの

分類結果近いデータの例正解率混同行列

ざっくり言うと

k近傍法は、今のデータに似ている過去のデータを探し、近くにあるデータの多数決で分類します。考え方はシンプルで、「近くの友だちに聞く」ような方法です。

合言葉

「近くにいる先輩データから予想する」

COLAB PRACTICE

Pythonで試してみる

Google Colabに貼り付けて動かせる練習コードです。まずはそのまま動かし、次に自分たちの列名に置き換えます。

COLAB / PYTHONコピーして試せる
# k近傍法：近いデータの多数決で分類する
from sklearn.datasets import load_iris
from sklearn.model_selection import train_test_split
from sklearn.neighbors import KNeighborsClassifier
from sklearn.metrics import accuracy_score, confusion_matrix

iris = load_iris(as_frame=True)
X = iris.frame[['petal length (cm)', 'petal width (cm)']]
y = iris.target

X_train, X_test, y_train, y_test = train_test_split(
    X, y, test_size=0.3, random_state=0
)

model = KNeighborsClassifier(n_neighbors=5)
model.fit(X_train, y_train)
pred = model.predict(X_test)

print('正解率:', accuracy_score(y_test, pred))
print('混同行列:')
print(confusion_matrix(y_test, pred))
