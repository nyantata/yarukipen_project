TECH CARD / LAB NOTE

可視化カード

数字の表を、発見できる形に変える

このカードの出番

  • ●データの傾向を最初に確認したい
  • ●センサ値と集中度の関係を見たい
  • ●発表で結果をわかりやすく伝えたい

作れるもの

折れ線グラフ散布図箱ひげ図ヒストグラム

ざっくり言うと

平均や標準偏差だけでは見えない「変化」「関係」「ばらつき」を、グラフで見えるようにします。まずは可視化。AIにかける前に、人間がデータを観察します。

合言葉

「数字の表を、発見できる形に変える」

イメージ図

流れを図で見る

flowchart LR A[CSVを読む] --> B[代表値を見る] B --> C[折れ線で時間変化] B --> D[散布図で関係] B --> E[箱ひげ図でばらつき] C --> F[発見をメモ] D --> F E --> F

HOW TO USE

使い方はこの順番

1

CSVをColabに読み込む

2

平均・最大・最小・標準偏差を確認する

3

目的に合うグラフを選ぶ

4

グラフから言えることと言えないことを分ける

COLAB PRACTICE

Pythonで試してみる

Google Colabに貼り付けて動かせる練習コードです。まずはそのまま動かし、次に自分たちの列名に置き換えます。

COLAB / PYTHONコピーして試せる
# まずはIrisデータで可視化の練習
import pandas as pd
import matplotlib.pyplot as plt
from sklearn.datasets import load_iris

iris = load_iris(as_frame=True)
df = iris.frame

display(df.head())
print(df.describe())

# 散布図：花びらの長さと幅の関係
plt.figure(figsize=(6,4))
plt.scatter(df['petal length (cm)'], df['petal width (cm)'])
plt.xlabel('petal length (cm)')
plt.ylabel('petal width (cm)')
plt.title('Iris scatter plot')
plt.show()

# ヒストグラム：がく片の長さの分布
plt.figure(figsize=(6,4))
plt.hist(df['sepal length (cm)'], bins=10)
plt.xlabel('sepal length (cm)')
plt.ylabel('count')
plt.title('Distribution')
plt.show()

OPEN DATA MISSION

オープンデータ演習

SSDSE-基本素材（SSDSE-E）をダウンロードし、都道府県ごとの人口・教育・医療・経済などから2つの列を選び、散布図を作る。グラフから「関係がありそう」「なさそう」を1文で説明する。

YARUKI LABで使うなら

YARUKI LABでは、心拍・振動・照度・音・主観スコアなどをグラフにして、プロダクトの根拠を探します。

オープンデータの入口

  • SSDSE（教育用標準データセット）：都道府県・市区町村などの公的統計をCSV/Excelで利用できます。
  • e-Stat：政府統計の総合窓口。CSVダウンロードやAPI利用ができます。
  • scikit-learn datasets：Irisなどの練習用データセットをColabですぐ使えます。
